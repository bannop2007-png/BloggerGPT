# BloggerGPT (статик + локальный прокси)

- Фронт: `index.html` (генераторы работают даже офлайн).
- Чат: работает только когда страница открыта через http://localhost:3000 и запущен сервер `npm run dev`.

## Шаги запуска
1) Убедись, что `.env` заполнен (ключи уже вставлены).
2) `npm install`
3) `npm run dev`
4) Открыть: http://localhost:3000/index.html

## Частые причины, почему "не работает"
- Открыли файл по `file://` — чат заблокирован. Нужен запуск через локальный сервер.
- Сервер не запущен или порт занят.
- Ключ/лимиты у провайдера.
